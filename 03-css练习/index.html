欢迎来到我的页面

这是一个简单的 HTML 页面示例，可直接部署到 GitHub Pages。

你可以按照以下步骤部署到 GitHub：

  1. 在项目目录下初始化 Git 仓库：git init
  2. 添加文件并提交：git add index.html && git commit -m "Initial commit"
  3. 在 GitHub 上创建新仓库
  4. 关联本地仓库和 GitHub 仓库：git remote add origin https://github.com/yyyjem/yuanchangweb--study.git
  5. 推送代码到 GitHub：git push -u origin main
  6. 在 GitHub 仓库的 Settings 里启用 GitHub Pages
